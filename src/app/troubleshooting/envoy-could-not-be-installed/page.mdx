# Envoy could not be installed
In this troubleshooting you will find out how to install Envoy on your own.

## Symptoms
- This error occurs during CLI setup: `failed to install Envoy: failed to execute command 'apt-get update': exit status 100`

## Resolution
1. Skip the envoy setup in the CLI
2. Follow the step-by-step installation on the [official envoy website](https://www.envoyproxy.io/docs/envoy/v1.32.2/start/install)
